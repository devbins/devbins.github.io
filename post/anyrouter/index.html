前言

Claude的模型在编程方面非常强，奈何非常贵，并且还要有一个好的网络才能用。

不过最近有anyrouter可以白嫖，也不需要魔法就能使用，非常方便。

不管如何先把账号注册了，万一后面没白嫖机会了。

记得上次 DeepSeek 就后悔没有早点白嫖，白白错过了。

注册账号

打开 https://anyrouter.top/register?aff=CuUY ，使用 GitHub 登录，这个链接可额外获得 $50 额度。

创建 Token

注册好后，先来创建token，后面要用到

找到API令牌，点击添加令牌，出现如下页面

token名字随便填，过期时间可以根据自己的需要修改

安装

以 Mac 为例

1
2

brew install node # claude code 需要 node 环境
npm install -g @anthropic-ai/claude-code # 安装到全局

查看安装是否成功

1
2

claude --version
1.0.44 (Claude Code)

有版本号说明安装成功

使用

1
2
3
4

cd project # 进入项目
export ANTHROPIC_AUTH_TOKEN=sk-... # 改成刚刚创建的 token
export ANTHROPIC_BASE_URL=https://anyrouter.top
claude # 启动claude
