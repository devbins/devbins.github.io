使用 GKD 跳过开屏广告

前言

每次打开应用都要等待几秒的广告，有些跳过还不好点，容易误触，今天教你使用 GKD 自动跳过广告。

GKD

GKD 是一款基于无障碍，高级选择器，订阅规则的自定义屏幕点击 Android 应用 它可以帮助你简化一些重复的流程, 如某些软件自动确认电脑登录 某些软件可能在启动时存在一些烦人的流程, 这个软件可以帮助你点击跳过这个流程

使用

下载

  1. GooglePlay
  2. GitHub Release Release v1.10.3 · gkd-kit/gkd

下载好好后进行安装

授权

安装好后打开应用，找到服务状态，并启用。

点击启用后，还需要授权，我这里选择的是手动授权，这里可以根据自己的情况选择合适的。

选择授权后，会跳转到无障碍，点击已下载的应用

打开使用“GKD”

这里会有危险的提示，点击确定同意（需等待 9s）

订阅

授权好后还不能用，因为 GKD 默认不提供规则，要自己添加

点击底部的订阅 tab 切换到订阅，然后点击 +

在弹出的对话框中输入订阅地址

我这里用的是 https://registry.npmmirror.com/@aisouler/gkd_subscription/latest/files/dist/AIsouler_gkd.json5

更多规则可以到 gkd-subscription · GitHub Topics 查看。

添加好后记得启用规则，这时候打开应用就可以跳过广告了。
