名字的由来

Hello 我是 dev.bins ，我的名字分为两部分，一部分是 dev 它是 developer 的缩写，并且它同时也是 /dev ，也就是 Linux 下外部设备的目录。

至于 bin 则是个人名字中的一个字的拼音，同时它也是 Linux 下的一个目录，但是发现 dev.bin 已经被人占用了，所以我就从名字中又取了一个字的拼音的首字母，恰好又是 Linux 下的目录，所以这两部分相结合就是我名字的由来了。

如果用全路径表示的话就是 /dev/bins ，所以你知道要从哪里找到我了。😜

它既能表达我是一个开发者，又能和我本人产生联系，确实有点奇妙。我非常喜欢这个名字。

Emacser

Emacs 死忠粉， Emacs 是我最喜欢的编辑器，早期我是 VIM 用户，不过现在我加入了 Emacs 大营，受 VIM 按键的影响，我在 Emacs 中也是使用 VIM 的键位。借助于 Evil 插件在 Emacs 中模拟 VIM ，让强者更强。

如果说 Emacs 是神的编辑器，而 VIM 则是编辑器之神，它们两者相结合，则是使用神的编辑器操作编辑器之神。

也就是 Spacemacs 的口号 The best editor is neither Emacs nor Vim, it's Emacs and Vim!

最喜欢的 Linux 发行版，Arch 系的 Manjaro

折腾过许多 Linux 发行版，包括 Ubuntu 、 CentOS 、 openSUSE 、 Arch 、 Manjaro 、 Deepin 、 Mint 最终选择了 Manjaro 作为我的主要桌面开发环境。
