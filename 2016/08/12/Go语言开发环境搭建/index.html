Go语言开发环境搭建

Go
  • Go

Go语言简介

Go 是一门开源，在语言层上支持并发，带垃圾回收的新型编译型语言。Go 语言是 Google 开发的，其性能可以与 C/C++ 媲美。具有动态语言的开发效率，静态语言的安全性。

安装

Mac 下安装

  • 使用homebrew安装，如果不知道homebrew是什么，请使用搜索解决。

Linux（Ubuntu）下安装

  1. 下载安装包下载地址
    在下载地址中根据自己电脑的系统选择对应的下载
  2. 解压下载的安装包到/usr/local

如果提示没有权限在最前面加上sudo

配置环境变量

  1. 编辑vi ~/.bashrc，如果没有自己手动创建。在文件的末尾加入下面的语句。
  • GOROOT是Go语言的安装目录
  • GOBIN是Go语言二进制文件目录,把GOBIN加入到PATH中可以在终端中使用TAB补齐
  • GOPATH是Go语言的工作目录，也就是我们通常开发用到的目录.从 go 语言 1.8 开始，GOPATH 默认为$HOME/go。

  2. 验证是否安装成功

GOPATH 结构

  • GOPATH 是 Go 语言的工作目录，在 GOPATH 下有三个重要的子目录分别是src、pkg、bin。
  • src存放 Go 语言的源代码
  • pkg存放编译后生成的文件
  • bin存放编译后生成的可执行文件

GoSublime 是 Sublime Text 3 的一个插件，装上该插件可以有代码提示的功能。
